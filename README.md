# Rasu Eaglercraft Mods

Eaglercraft向けのJavaScript Mod集です。

このリポジトリでは、共通APIである **RasuModAPI** と、それを利用した各種Modを公開しています。

## Files

### `RasuModAPI.js`

すべてのModで使用する共通APIです。

### `japaneseIME.js`

Eaglercraftで日本語入力（IME）をサポートするModです。

## Usage

ここで配布しているModを利用する際には必ず`RasuModAPI.js` を先に読み込み、その後に使用したいModを読み込んでください。
